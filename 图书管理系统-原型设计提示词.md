# 图书管理系统 - 原型设计提示词

## 说明
本文档提供详细的页面原型设计提示词，可用于：
1. Figma、即时设计等设计工具的AI设计功能
2. 向设计师描述页面需求
3. 前端开发人员的页面布局参考

---

## 设计规范

### 配色方案
- **主色调**: #409EFF（Element Plus默认蓝色）
- **成功色**: #67C23A（绿色）
- **警告色**: #E6A23C（橙色）
- **错误色**: #F56C6C（红色）
- **信息色**: #909399（灰色）
- **背景色**: #F5F7FA（浅灰）
- **文字主色**: #303133（深灰）
- **文字次色**: #909399（中灰）

### 字体规范
- **大标题**: 24px, 粗体
- **中标题**: 18px, 粗体
- **小标题**: 16px, 粗体
- **正文**: 14px, 常规
- **辅助文字**: 12px, 常规

### 间距规范
- **大间距**: 40px
- **中间距**: 24px
- **小间距**: 16px
- **最小间距**: 8px

### 圆角规范
- **卡片圆角**: 8px
- **按钮圆角**: 4px
- **输入框圆角**: 4px

## 一、前台读者门户

### 1.1 登录页面

**提示词：**
```
设计一个现代简洁的图书管理系统登录页面。页面整体采用左右分栏布局，左侧占40%宽度，右侧占60%宽度。

左侧区域：
- 背景色为渐变蓝色（#409EFF到#66B1FF），全高度
- 垂直居中显示一个白色图书图标（简约线条风格）和系统名称"图书管理系统"
- 下方显示slogan："让知识触手可及"

右侧区域：
- 纯白色背景
- 垂直居中显示登录表单
- 表单卡片：
  - 宽度400px，居中显示
  - 顶部显示"欢迎登录"标题（24px粗体）
  - 用户名输入框：带图标（用户头像图标），placeholder"请输入用户名/手机号/邮箱"
  - 密码输入框：带图标（锁图标），placeholder"请输入密码"，支持显示/隐藏切换（眼睛图标）
  - "记住我"复选框（左侧）和"忘记密码？"链接（右侧，蓝色文字）在同一行
  - 登录按钮：宽度100%，蓝色背景，白色文字，高度40px
  - 底部文字："还没有账户？立即注册"（注册为蓝色链接）

整体风格：现代、简洁、专业，使用柔和的阴影效果，圆角设计
```

### 1.2 注册页面

**提示词：**
```
设计一个图书管理系统用户注册页面。采用与登录页面相同的左右分栏布局。

左侧区域：与登录页面相同

右侧区域：
- 纯白色背景
- 垂直居中显示注册表单
- 表单卡片：
  - 宽度450px，居中显示
  - 顶部显示"新用户注册"标题（24px粗体）
  - 表单字段（每个字段带对应图标）：
    1. 用户名输入框（用户图标）
    2. 真实姓名输入框（身份证图标）
    3. 手机号输入框（手机图标），右侧带"发送验证码"按钮（60秒倒计时）
    4. 验证码输入框（盾牌图标）
    5. 密码输入框（锁图标），带强度指示器（弱/中/强，不同颜色）
    6. 确认密码输入框（锁图标）
  - 用户协议复选框："我已阅读并同意《用户协议》和《隐私政策》"
  - 注册按钮：宽度100%，蓝色背景，白色文字，高度40px
  - 底部文字："已有账户？立即登录"（登录为蓝色链接）

表单字段间距16px，字段高度40px
```

### 1.3 首页

**提示词：**
```
设计一个现代风格的图书管理系统首页。

顶部导航栏（固定高度60px，白色背景，下方有浅灰色阴影）：
- 左侧：系统Logo和名称"图书管理系统"
- 中间：导航菜单（首页、图书检索、图书分类、我的借阅）
- 右侧：搜索框（宽度300px）+ 用户头像下拉菜单（显示用户名、头像、下拉显示个人中心/退出登录）

轮播Banner区域（高度400px）：
- 3-4张图书相关的精美banner图
- 左右切换箭头
- 底部圆点指示器

快速搜索区域（位于Banner下方，居中显示）：
- 大号搜索框（宽度800px，高度50px）
- 搜索框左侧有图标，placeholder"搜索书名、作者、ISBN..."
- 搜索按钮在右侧
- 搜索框下方显示热门搜索词标签（可点击）

图书分类卡片区域：
- 标题"图书分类"（18px粗体）
- 网格布局展示分类（4列，每行4个分类）
- 每个分类卡片：
  - 卡片背景色（不同分类不同的浅色背景）
  - 分类图标
  - 分类名称
  - 图书数量
  - 鼠标悬停有阴影效果和轻微放大

推荐图书区域：
- 标题"热门推荐"（18px粗体）+ "查看更多"链接
- 横向滚动图书列表（展示8-10本图书）
- 每本图书卡片：
  - 图书封面（高度200px）
  - 书名（最多显示2行，超出显示省略号）
  - 作者
  - 可借状态标签（绿色"可借"或灰色"已借完"）
  - 鼠标悬停有卡片上浮效果

页面底部（灰色背景）：
- 系统介绍、联系方式
- 版权信息

整体采用卡片式设计，间距合理，清爽简洁
```

**提示词：**
```
设计一个功能完善的图书检索页面。

顶部导航栏：与首页相同

页面布局：左侧筛选栏 + 右侧内容区域

左侧筛选栏（宽度250px，浅灰色背景）：

  - 书名输入框
  - 作者输入框
  - ISBN输入框
  - 出版社输入框
  - 分类选择（树形结构，可展开/收起）
  - 出版年份范围（开始年份-结束年份）
  - 可借状态（单选：全部/可借/已借完）
- 底部：重置按钮 + 搜索按钮

右侧内容区域：
- 顶部工具栏：
  - 左侧：搜索结果统计"共找到XXX本图书"
  - 中间：排序方式下拉框（相关度/书名/出版日期）
  - 右侧：视图切换按钮（列表视图/网格视图）

图书列表（列表视图）：
- 每个图书项（白色卡片，鼠标悬停有浅蓝色边框）：
  - 左侧：图书封面缩略图（100x140px）
  - 中间：
    - 书名（18px粗体，蓝色可点击）
    - 作者、出版社、出版日期（灰色文字）
    - 简介（最多显示2行）
  - 右侧：
    - 可借状态标签
    - 馆藏信息"馆藏: X本，可借: X本"
    - 查看详情按钮

图书网格（网格视图）：
- 4列布局
- 每个图书卡片：
  - 图书封面（宽度100%）
  - 书名（最多2行）
  - 作者
  - 可借状态标签
  - 查看详情按钮

底部分页器：
- 居中显示
- 显示总页数、当前页
- 上一页、页码、下一页、跳转输入框
```

- 页面标题"当前借阅"（24px粗体）

  - 当前借阅数：X本
  - 可借数量：X本
  - 逾期数量：X本（红色）

  - 表格形式展示

  - 文字："暂无借阅记录"
