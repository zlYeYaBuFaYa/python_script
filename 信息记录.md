## git 操作:

- 1. 创建 SSH Key

```
git config --global user.name
git config --global user.email


1. 安装Git	在WSL中安装Git软件包	sudo apt update && sudo apt install git
2. 配置身份	设置全局用户名和邮箱	git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"
3. 生成密钥	创建SSH密钥对	ssh-keygen -t rsa -b 4096 -C "你的邮箱"
4. 添加公钥	将公钥内容复制到Git服务商网站	1. 复制：cat ~/.ssh/id_rsa.pub
2. 粘贴到Git服务商的SSH密钥设置页面
5. 测试连接	验证SSH连接是否成功	ssh -T git@github.com (以GitHub为例)
```

- 2. git bash 中文乱码:

```
git config --global core.quotepath false
```

- 3. 关联远程仓库:

```
git remote add origin git@github.com:username/repository.git
git remote add set-url git@github.com:zlYeYaBuFaYa/go-demo3.git
git push --set-upstream set-url main

git remote rename oldname newname # 重命名远程仓库别名
git remote -v # 查看远程仓库信息

注: set-url 是自定义的别名，可以随意取名，main 是远程分支名
```

## Window 设置 wsl2 端口转发:

```
1. 设置端口转发
netsh interface portproxy add v4tov4 listenport=22 listenaddress=0.0.0.0 connectport=22 connectaddress=172.19.21.232

2. 查看端口转发
netsh interface portproxy show all

3. 删除端口转发
netsh interface portproxy delete v4tov4 listenport=22 listenaddress=0.0.0.0

export http_proxy=http://192.168.1.2:22
```

## mac homebrew 安装

```
1. homebrew文档连接: https://www.cnblogs.com/it-feiyu/p/18647727

2. 安装Command Line Tools： xcode-select --install

3. 国内镜像安装: /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

4. git 克隆安装:sudo git clone https://github.com/Homebrew/brew /opt/homebrew

5. 清华大学镜像:
# 替换 brew.git
git -C "$(brew --repo)" remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git

# 替换 homebrew-core.git
git -C "$(brew --repo homebrew/core)" remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git

# 替换 homebrew-cask.git
git -C "$(brew --repo homebrew/cask)" remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-cask.git


6. 验证安装: brew --version # 查看 Homebrew 版本
```
